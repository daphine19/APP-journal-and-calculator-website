About APP

Key features of the assignment

Calculator

This part of the website allows a user carry out basic calculation ie addition, subtraction, multiplication and division. This can be done both via the server and client side.

Journal

Displays a list of journal entries that were stored in a json file. And if a user clicks on that entry it is seen under the selected entry part. Furthermore, the user can delete an entry and also add one to the list. finally the list can be uploaded to the server via the upload button.

Design

The design was inspired by aesthetic colour combination and a some knowledge from various platforms. Completing this section was backed up with PARC(Proximity, Alignment, Repetition, Contrast) knowledge

Thoughts

Works similarly like the journal section though intended mainly for the page designer. the only addition is that it displays the thought on the page each time the generate thought button is clicked.
